# cuRobo — Jerk Minimization YAML Schema

**Role in this project:** JERK MINIMIZATION ONLY. Not path planning. Not obstacle avoidance.
Removes mechanical artifacts (motor cogging, cable drag, gearbox backlash) from
6-axis cinema robot arm trajectories via joint-space jerk minimization (TrajOpt).

---

## CRITICAL: Forbidden Parameters

The following parameters MUST NOT appear in any cuRobo config generated for this project:

```
❌ collision_spheres
❌ world_model
❌ path_plan
❌ obstacle_avoidance
❌ obstacle_*
❌ scene_collision
❌ voxel_*
```

If these appear, the validator rejects the config as a scope violation.

## cuRobo Jerk Config Schema

```yaml
# curob_jerk_config.yaml
# Generated by curob_config agent — cinema robot jerk minimization

robot_cfg:
  kinematics:
    usd_path: "/registry/usd/robot_7.usd"        # from USD registry
    usd_robot_root: "/World/Robot"
    base_link: "base_link"
    ee_link: "end_effector"
    robot_name: "cinema_robot_7"

    # Joint names — must match URDF joint names exactly
    joint_names:
      - "j1_pan"
      - "j2_tilt"
      - "j3_roll"
      - "j4_pitch"
      - "j5_yaw"
      - "j6_roll_end"

    # Per-joint velocity limits (rad/s) — from empirical DB
    # NULL joints: omit from list or set to null
    joint_velocity_limits:
      j1_pan: 2.094395          # rad/s (from DB: 120 deg/s)
      j2_tilt: 1.745329         # rad/s (100 deg/s)
      j3_roll: 2.617994         # rad/s (150 deg/s)
      j4_pitch: 3.141593        # rad/s (180 deg/s)
      j5_yaw: 3.141593          # rad/s (180 deg/s)
      j6_roll_end: 6.283185     # rad/s (360 deg/s)

    # Per-joint acceleration limits (rad/s²) — from empirical DB
    joint_acceleration_limits:
      j1_pan: 5.235988          # rad/s²
      j2_tilt: 4.363323
      j3_roll: 7.853982
      j4_pitch: 10.471976
      j5_yaw: 10.471976
      j6_roll_end: 20.943951

    # Per-joint jerk limits (rad/s³) — PRIMARY OUTPUT of this agent
    # Computed from velocity/acceleration if not directly specified in DB
    # NULL if velocity_limit or acceleration_limit is NULL
    joint_jerk_limits:
      j1_pan: 15.707963         # rad/s³
      j2_tilt: 12.566371
      j3_roll: 25.132741
      j4_pitch: 31.415927
      j5_yaw: 31.415927
      j6_roll_end: 62.831853

    # Position limits (radians) — from empirical DB
    joint_limits:
      j1_pan:  {lower: -3.14159, upper: 3.14159}
      j2_tilt: {lower: -1.57080, upper: 1.57080}
      j3_roll: {lower: -3.14159, upper: 3.14159}
      j4_pitch: {lower: -2.09440, upper: 2.09440}
      j5_yaw: {lower: -3.14159, upper: 3.14159}
      j6_roll_end: {lower: -6.28318, upper: 6.28318}

# TrajOpt settings — jerk minimization
trajopt_cfg:
  num_trajopt_seeds: 4
  num_graph_seeds: 0               # no graph planning (jerk-only mode)

  # Jerk minimization objective weights
  trajopt_cost_weights:
    jerk_weight: 1.0               # PRIMARY: minimize joint-space jerk
    velocity_weight: 0.1           # secondary: penalize high velocities
    acceleration_weight: 0.2       # secondary: penalize high accelerations
    position_weight: 0.0           # do not optimize position (preserve path)

  # Smoothing parameters
  smooth_weight:
    jerk: [10.0, 10.0, 10.0, 10.0, 10.0, 10.0]  # per-joint (or single value)
    acceleration: [0.5, 0.5, 0.5, 0.5, 0.5, 0.5]

  # Convergence
  max_iterations: 200
  convergence_threshold: 1.0e-4

# Processing mode
processing:
  mode: "batch"                    # "batch" | "online"
  # batch: process full trajectory offline, best quality
  # online: process in real-time, lower latency, less smoothing

# Input/Output
io:
  input_format: "joint_trajectory"  # joint angles over time
  output_format: "joint_trajectory"  # smoothed joint angles over time
  time_resolution: 0.01            # seconds between waypoints (100Hz)
```

## Python API

```python
import torch
from curobo.types.robot import RobotConfig
from curobo.wrap.reacher.trajopt import TrajOptSolver

# Load config
robot_cfg = RobotConfig.from_dict({...})  # or from YAML

# Initialize TrajOpt solver (jerk minimization mode)
solver = TrajOptSolver.from_yaml("curob_jerk_config.yaml")

# Input: raw joint trajectory (T, N_joints)
raw_trajectory = torch.tensor(raw_joint_angles)  # (T, 6) float32

# Smooth: minimize jerk while preserving path intent
result = solver.solve(
    goal=raw_trajectory[-1],     # terminal state preserved
    waypoints=raw_trajectory,    # all waypoints as soft constraints
)

# Output: smoothed trajectory
smooth_trajectory = result.solution  # (T, 6) float32
jerk_profile = result.metrics.jerk  # per-joint jerk over time
```

## Standard Launch Pattern

```python
# ros2/robot_launch.py
import launch
from launch import LaunchDescription
from launch_ros.actions import ComposableNodeContainer, Node
from launch_ros.descriptions import ComposableNode

def generate_launch_description():
    return LaunchDescription([

        # Isaac ROS 4.0 node graph for cinema robot
        ComposableNodeContainer(
            name="robot_container",
            namespace="",
            package="rclcpp_components",
            executable="component_container_mt",  # multi-threaded
            composable_node_descriptions=[

                # Robot state publisher
                ComposableNode(
                    package="robot_state_publisher",
                    plugin="robot_state_publisher::RobotStatePublisher",
                    parameters=[{"robot_description": urdf_content}],
                ),

                # Joint state controller
                ComposableNode(
                    package="isaac_ros_joint_control",
                    plugin="nvidia::isaac_ros::joint_control::JointControlNode",
                    parameters=[{
                        "joint_names": ["j1_pan","j2_tilt","j3_roll","j4_pitch","j5_yaw","j6_roll_end"],
                        "control_mode": "position",  # "position" | "velocity" | "effort"
                    }],
                    remappings=[
                        ("joint_commands", "/robot/joint_commands"),
                        ("joint_states", "/robot/joint_states"),
                    ],
                ),
            ],
        ),
    ])
```
